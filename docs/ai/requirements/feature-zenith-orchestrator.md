# Feature: Zenith Ultimate Orchestrator (ZUO) - The World-Class Business Command Center

## Problem Statement
1. **Static CRM Data:** Current CRM systems are passive data repositories. They wait for users to enter data and only show what has happened, not what *should* happen next to win the market.
2. **Generic UI/UX:** Standard business tools lack the "Premium feel" that inspires high-performance leaders. They feel like utility tools rather than a powerful cockpit for a business.
3. **Information Overload:** Leaders are buried in lists. They need a system that distills complex pipeline data into "Actionable Orbs" and clear strategic mandates.
4. **Isolated Business Intelligence:** Strategy and execution (deals) are often separated. The system needs to bridge the gap between "Market Intelligence" and "Daily Tasks".

## Goals
- **Zenith AI Command (Orchestrator):** A primary dashboard mode that proactive suggests business pivots and deal-specific closing tactics.
- **Hyper-Premium Aesthetic:** Implement "Glassmorphism 3.0" with deep obsidian backgrounds, gold accents (Success), and neon cyber-highlights for urgency.
- **Predictive Performance Engine:** AI that calculates "Win Probability" based on behavioral patterns (last activity, note sentiment, deal velocity).
- **Actionable Notification Orbs:** Floating, interactive notification nodes that allow instant task execution (e.g., "Approve Discount" or "Send Follow-up") without navigating away.
- **Dynamic UX Morphing:** The UI layout shifts its "Gravity" toward the most important task of the hour (Focus Mode).
- **System-wide Micro-interactions:** Haptic-like visual feedback on every button, smooth layout transitions, and meaningful animations for "Won" states.

## Non-Goals
- Building a full ERP or supply chain management system.
- Integrating with external physical hardware or non-digital business processes.
- Replacing the core database (Supabase) but rather enhancing how it is interacted with.

## User Stories
- As a CEO, I want the system to present a "Morning Briefing" in a stunning visual format so I can see the entire organization's health in 60 seconds.
- As a Sales Leader, I want to use "Focus Mode" which hides all low-priority deals and highlights only the multi-million THB opportunities that need closing today.
- As a user, I want the buttons to feel "Weighty" and "Premium" through advanced CSS effects, making the software feel expensive and high-quality.
- As a user, I want AI to draft the "Perfect Email" based on the last 3 notes in a deal, so I can communicate with high impact and zero effort.
- As a user, I want notifications to be "Orbs" that I can swipe or click directly on the screen to take immediate action.

## Success Criteria
- [ ] Implement a "High-Performance Theme" switch (Dark/Glass) that transforms the entire app's look and feel to a premium professional grade.
- [ ] AI Command Center provides at least 3 distinct "Strategic Mandates" per day based on real-time data.
- [ ] Interactive "Notification Orbs" are functional for common tasks (Notes/Tasks).
- [ ] All primary buttons and cards utilize a unified design system (Neumorphism + Glassmorphism).
- [ ] Page transitions and card interactions use framer-motion or CSS transitions for a "Silky" feel.

## Constraints and Assumptions
- Use Vanilla CSS and standard React hooks for maximum control over aesthetics.
- Gemini API continues to be the primary engine for AI intelligence.
- The system must remain fast and responsive despite high-end visual effects (performance optimization is key).
